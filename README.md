# autodsm-web-landing

Public marketing site for [AutoDSM](https://github.com/AutoDSM)—design system intelligence from your repository.

## Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
